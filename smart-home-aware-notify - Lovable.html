Utilizing a Passive Infrared (PIR) sensor, the system detects the presence of individuals in a room, while a current sensor continuously monitors the operational status of the appliance. The mod implementation of this smart appliance control system not only promotes energy efficiency by reducing unnecessary electricity consumption but also enhances user convenience through real-time monitoring and control capabilities. For the hardware implementation, here's the Arduino code you would need to upload to your ESP8266 (this would be saved in your Arduino IDE):

#include <ESP8266WiFi.h>
#include <ESP8266WebServer.h>

// WiFi credentials
const char* ssid = "YourWiFiSSID";
const char* password = "YourWiFiPassword";

// Pin definitions
const int PIR_PIN = D1;           // PIR sensor
const int CURRENT_SENSOR_PIN = A0; // Current sensor
const int RELAY_PIN = D2;         // Relay control

ESP8266WebServer server(80);
bool isApplianceOn = false;
bool occupancyDetected = false;

void setup() {
  Serial.begin(115200);
  
  // Configure pins
  pinMode(PIR_PIN, INPUT);
  pinMode(RELAY_PIN, OUTPUT);
  digitalWrite(RELAY_PIN, LOW);
  
  // Connect to WiFi
  WiFi.begin(ssid, password);
  while (WiFi.status() != WL_CONNECTED) {
    delay(500);
    Serial.print(".");
  }
  
  // Setup API endpoints
  server.on("/status", HTTP_GET, handleGetStatus);
  server.on("/toggle", HTTP_POST, handleToggleAppliance);
  
  server.begin();
  Serial.println("HTTP server started");
  Serial.println(WiFi.localIP());
}

void loop() {
  server.handleClient();
  
  // Read sensors
  occupancyDetected = digitalRead(PIR_PIN);
  float current = readCurrent();
  
  // Check for unsafe condition
  if (isApplianceOn && !occupancyDetected) {
    // You could implement additional logic here
    Serial.println("Warning: Appliance running with no occupancy");
  }
  
  delay(100);
}

float readCurrent() {
  int sensorValue = analogRead(CURRENT_SENSOR_PIN);
  // Convert analog reading to amperes (calibration needed)
  return sensorValue * (5.0 / 1024.0);
}

void handleGetStatus() {
  String json = "{";
  json += "\"isApplianceOn\":" + String(isApplianceOn);
  json += ",\"occupancyDetected\":" + String(occupancyDetected);
  json += ",\"current\":" + String(readCurrent());
  json += "}";
  
  server.send(200, "application/json", json);
}

void handleToggleAppliance() {
  isApplianceOn = !isApplianceOn;
  digitalWrite(RELAY_PIN, isApplianceOn ? HIGH : LOW);
  
  server.send(200, "application/json", "{\"success\":true}");
}

A 5V power supply is an electrical source that provides 5 volts of direct current (DC) electricity, which is commonly used to power microcontrollers like the ESP8266, sensors, and other low-voltage electronic components in IoT and embedded systems.

For this specific IoT project, a 5V power supply would typically:

  • Power the ESP8266 microcontroller
  • Provide electricity to the PIR sensor
  • Supply voltage to the relay module
  • Support the current sensor

There are several ways to provide 5V:

  1. USB power adapter
  2. Power bank
  3. Dedicated 5V DC power supply
  4. Battery pack with voltage regulator
  5. Computer USB port

In the Arduino code (SmartHomeDevice.ino), the power supply would connect to:

  • VIN pin for ESP8266
  • VCC pins for sensors
  • Power rail for relay module
